Evènements : International

Le monde de la tech est global ! Rester connecté aux événements internationaux est crucial pour comprendre les grandes tendances, découvrir des technologies émergentes et s'inspirer des leaders de l'industrie. Même si une participation physique est souvent difficile, de nombreuses conférences proposent des diffusions en ligne ou des retranscriptions, accessibles aux étudiants et au staff.

Google I/O (États-Unis)

Description : La conférence annuelle des développeurs de Google. C'est l'événement majeur pour toutes les annonces concernant Android, Chrome, les services Google Cloud, l'IA, et les nouvelles plateformes de développement.

  • Thèmes : Android, Flutter, Cloud, IA/Machine Learning, Web, ChromeOS, Assistant Google.
  • Public : Développeurs, chefs de produit, designers, fans de Google.
  • Période : Généralement en mai.
  • Comment suivre : Diffusions en direct et vidéos disponibles sur la chaîne YouTube Google Developers après l'événement.
  • Site web : io.google/2025/ (lien pour l'édition 2025, à adapter)

Apple WWDC (Worldwide Developers Conference - États-Unis)

Description : La conférence d'Apple dédiée aux développeurs. C'est là que sont présentées les nouveautés des systèmes d'exploitation (iOS, macOS, watchOS, tvOS) et les nouveaux outils pour les développeurs sur l'écosystème Apple.

  • Thèmes : iOS, macOS, Swift, Xcode, App Store, réalité augmentée/virtuelle.
  • Public : Développeurs Apple, étudiants en développement mobile.
  • Période : Généralement en juin.
  • Comment suivre : Keynote accessible en direct et sessions techniques disponibles via l'application Apple Developer ou le site web.
  • Site web : developer.apple.com/wwdc/

re:Invent (AWS - États-Unis)

Description : La conférence annuelle d'Amazon Web Services (AWS) pour le cloud computing. Des milliers de sessions techniques, des ateliers pratiques et des annonces majeures sur les services cloud.

  • Thèmes : Cloud computing, IA/ML sur AWS, bases de données, sécurité cloud, serverless, DevOps.
  • Public : Ingénieurs cloud, architectes système, développeurs backend, DevOps.
  • Période : Généralement fin novembre / début décembre.
  • Comment suivre : De nombreuses sessions sont enregistrées et disponibles sur la chaîne YouTube AWS après l'événement.
  • Site web : reinvent.awsevents.com

KubeCon + CloudNativeCon (Divers lieux dans le monde)

Description : La conférence phare de la Cloud Native Computing Foundation (CNCF). Elle rassemble des développeurs et des utilisateurs autour des technologies cloud-native comme Kubernetes, Prometheus, et les microservices.

  • Thèmes : Conteneurisation (Docker, Kubernetes), microservices, CI/CD, Observability, DevOps.
  • Public : Ingénieurs DevOps, architectes logiciels, développeurs backend.
  • Période : Plusieurs éditions par an (Amérique du Nord, Europe, Chine).
  • Comment suivre : De nombreuses sessions sont disponibles en ligne.
  • Site web : kccnc.io
